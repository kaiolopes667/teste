Coruja - Mobit

O projeto tem o objetivo de desenvolver um painel (software de monitoramento) digital inteligente em nuvem que possibilite o monitoramento de dispositivos de fiscalização eletrônica para faixas monitoradas de diferentes fabricantes.

EQUIPE: Carlos Henrique Leitao Cavalcante, Ajalmar Rêgo da Rocha Neto, Daniel Alencar Barros Tavares, Jefferson Calixto Figueiredo, Carlos Anderson Ferreira Sales, Elizabeth Lucas Bruno, Luan Sousa Cordeiro, Lucas Silva de Sousa, Ronaldo Tadeu Pontes Milfont, Luid Sousa da Silva, Antonio Iedo Rocha Ribeiro.

Direct - Jornal Opovo

O objetivo do projeto é o desenvolvimento de um Sistema Inteligente de Identificação de preferências de usuários, formado por um conjunto de módulos que devem trabalhar em conjunto para que se possa obter de forma automática a alocação de conteúdo e publicidade de acordo com as preferências de acesso do usuário que está navegando no site da empresa.

EQUIPE: Carlos Henrique Leitao Cavalcante, Alisson Gomes Linhares, Ajalmar Rêgo da Rocha Neto, Daniel Alencar Barros Tavares, Aline Santos Ferreira, Alan Lucas Silva Matias, Elizabeth Lucas Bruno, Luan Sousa Cordeiro, Amauri Holanda de Souza Júnior, Lucas Silva de Sousa, José Alberth Vasconcelos Florêncio, Eduardo Mariano de Barros, Gleysson Rocha Luiz, Francisco Felipe Moreira Sousa, Claudiane Gomes Ferreira.

Overhead - Boa Vista Tecnologia

O projeto consiste em uma solução computacional em nuvem que possibilite, através de uma interface wizard web, criar uma camada capaz de automatizar e facilitar a configuração e a extração de dados de fontes web que não forneçam uma forma de exportar as informações das movimentações dos seus clientes. A necessidade dessa solução é para diminuir o OVERHEAD do esforço de configuração de novos clientes. Dessa forma, o projeto tem como objetivo criar um sistema em nuvem capaz de configurar fontes de dados web, realizar a captura automática de informações através do uso de robôs computacionais que utilizam Web Scraping e disponibilizá-los através de uma API que proverá comunicação com os sistemas da empresa parceira.

EQUIPE: Carlos Henrique Leitao Cavalcante, Alisson Gomes Linhares, Ajalmar Rêgo da Rocha Neto, Elizabeth Lucas Bruno, Lucas Silva de Sousa, Davi Alves Leão, Eduardo Mariano de Barros, Ana Ádila Colares Costa, Daniel Ferreira do Nascimento, Thiago Simão Sales, Paulo Ricardo Bernardo Silva, Gabriel Marinho Santos de Sousa, João Pedro Silva de Sousa, Laryssa Lima Coutinho.
